# Audio_transcriber
**Tamil Voice Translator**
A real-time Tamil Voice Translator that converts spoken Tamil into multiple target languages
instantly.
The application combines speech recognition and machine translation with a modern, user-friendly
interface.
**Key Features**
• Real-time Tamil Speech Recognition
• Multi-language translation (English, Hindi, Spanish, German, Japanese)
• High-speed processing
• One-click microphone recording
• Clear text functionality
• Powered by MyMemory Translation API
**Workflow**
1. Select target languages
2. Click the microphone button
3. Speak clearly in Tamil
4. View recognized text and translations instantly
